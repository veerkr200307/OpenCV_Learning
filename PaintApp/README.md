# OpenCV Paint Application

A simple interactive paint application built using Python and OpenCV.

## Features

- Draw using the mouse
- Change brush color
- Increase/decrease brush size
- Clear canvas
- Save drawing
- Quit application

## Controls

| Key | Action |
|------|--------
| G | Green Brush |
| R | Red Brush |
| E | Eraser |
| B | Blue Brush |
| + | Increase Brush Size |
| - | Decrease Brush Size |
| C | Clear Canvas |
| S | Save Drawing |
| X | Exit |

## Technologies

- Python
- OpenCV
- NumPy
